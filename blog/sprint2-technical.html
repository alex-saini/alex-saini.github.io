What is the difference between Margin, Border, and Padding?

The Box Model

The Box Model is used in CSS to help layout and design certain aspects of your page. It helps make the layout of the page simple by placing everything into, you guessed it 'boxes'. The outermost box is the margin, then coming inwards goes the border, padding and then your content. The content is the centermost part of the box, holding the html. The padding surrounds the content and allows you to customize all four sides, making them bigger or smaller. The border sits within the padding and margin, and by default it is invisible. You can customize the border with attributes such as the colour and thickness for design aspects/layout. The margin is the outermost box and seperates everything that is inside this box model with other CSS box models within the same page.
